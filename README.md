# RustyNotes

A rust and iced markdown note taking app.

## Roadmap

- [ ] Markdown editor with syntax highlighting
- [ ] Edit/Live-Preview/View mode (like Obsidian)
- [ ] Handling of a full folder (like Obsidian)
- [ ] Handling of markdown and wiki-links
- [ ] Central managing of known folders in .cache/RustyNotes
- [ ] Per folder settings in .rusty inside the folder
- [ ] Fast opening
- [ ] Fast operations
- [ ] Choose fonts
- [ ] Select personal theme (.rusty/themes) (should be able to follow the night/day of os)
